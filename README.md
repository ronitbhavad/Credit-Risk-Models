# Credit Risk Modeling Portfolio

This repository contains credit risk modeling projects commonly used in banking risk management.

Projects Included:

1. Probability of Default (PD) Model
   - Logistic Regression
   - ROC Curve
   - Model evaluation metrics

2. Credit Scorecard Model
   - Weight of Evidence (WOE)
   - Information Value (IV)
   - Scorecard development

3. Expected Loss Model
   - Expected Loss = PD × LGD × EAD
   - Portfolio level loss estimation

4. Portfolio Credit Risk Simulation
   - Monte Carlo simulation
   - Portfolio loss distribution

Tools Used:
Python, Pandas, Scikit-learn, Matplotlib

Purpose:
To demonstrate practical implementation of credit risk models used in financial institutions.
